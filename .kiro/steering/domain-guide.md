
# Domain Guide — Spain Gestoría & Expat

## Key Processes
- Gestoría: Modelo 303, 111, IRPF, DGT transfers
- Expat: NIE/TIE, Digital-Nomad visas, Empadronamiento

## Policies
- Residency: data must remain in Spain-only mode when configured
- Redaction: DNI/passport numbers redacted in logs/traces
- Audit: all actions attributable to OIDC user and workspace

## Templates
- Store YAML blueprints in `templates/` (create folder when needed)
- Validation via `/templates/validate`
