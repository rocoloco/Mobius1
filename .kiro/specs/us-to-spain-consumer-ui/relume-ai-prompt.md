## Key Details to Emphasize

- **Generous whitespace** - This is critical for the "simple" philosophy
- **Progress indicators** - Users need to see where they are in processes
- **Status badges** - Visual feedback on document/workflow states
- **Card-based layouts** - Information grouped in digestible chunks
- **Clear CTAs** - Primary actions should be obvious (teal buttons)
- **Sidebar navigation** - Persistent navigation for easy access
- **Empty states** - Friendly illustrations when no content exists

## Color Palette Reference for Relume

Primary: Teal (#008B8B)
Accent: Coral (#FF7F50)
Secondary: Sandy Brown (#F4A460)
Text: Charcoal (#2F4F4F)

## Component Patterns to Request

- **Workflow Cards**: Icon at top, title, description, progress bar, metadata (steps, time), button
- **Document Cards**: Icon/thumbnail, name, metadata, status badge, action buttons
- **Progress Bars**: 8px height, rounded ends, teal fill on light gray background
- **Status Badges**: Pill-shaped, colored backgrounds (green=completed, blue=in progress, gray=pending, red=overdue)
- **Form Inputs**: 44px height, rounded corners, clear labels above, validation messages below
- **Buttons**: 44px height, 6px border radius, primary=teal, secondary=outlined
