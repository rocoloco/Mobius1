## Design Specifications

### Color Palette
- **Primary (Teal)**: #008B8B - Buttons, links, active states
- **Accent (Coral)**: #FF7F50 - Highlights, warnings, CTAs
- **Secondary (Sandy Brown)**: #F4A460 - Success states, progress
- **Neutral (Charcoal)**: #2F4F4F - Text, headings
- **Cream (Cornsilk)**: #FFF8DC - Backgrounds, cards

### Typography
- **Font**: Inter (sans-serif)
- **Headings**: Bold (700) to Semibold (600)
- **Body**: Normal (400)
- **Scale**: 12px to 60px

### Spacing
- **Base Unit**: 8px
- **Common Spacing**: 16px, 24px, 32px, 48px

### Components
- **Buttons**: 44px height, 6px border radius
- **Cards**: 8px border radius, subtle shadow
- **Inputs**: 44px height, 4px border radius
- **Modals**: 12px border radius, large shadow

## Design Philosophy

### Simple
- Generous whitespace (24px minimum between sections)
- Limited color palette (5 core colors)
- Clear visual hierarchy
- Minimal decorative elements

### Magical
- Smooth transitions and animations
- Delightful illustrations for empty states
- Smart defaults and recommendations
- Purposeful micro-interactions

### Functional
- High contrast for readability (WCAG AA)
- Touch-friendly targets (44px minimum)
- Clear affordances (buttons look clickable)
- Consistent patterns across all pages

## Accessibility

- **Color Contrast**: All text meets WCAG 2.1 AA standards (4.5:1 minimum)
- **Focus States**: Visible 2px focus indicators on all interactive elements
- **Touch Targets**: Minimum 44px Ã— 44px for all interactive elements
- **Semantic HTML**: Proper heading hierarchy and ARIA labels

## Responsive Breakpoints

- **Mobile**: < 768px (single column, stacked layout)
- **Tablet**: 768px - 1024px (2-column grid)
- **Desktop**: > 1024px (multi-column, full sidebar)

## Export Settings

- **Format**: PNG for mockups, SVG for icons
- **Scale**: 2x for retina displays
- **Naming**: `page-name-viewport.png`
- **Location**: This folder
