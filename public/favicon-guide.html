HTML Implementation

Add these tags to your <head> section:

<!-- SVG Favicon (modern browsers with theme support) -->
<link rel="icon" type="image/svg+xml" href="/assets/logos/favicon.svg">

<!-- Fallback for older browsers -->
<link rel="icon" type="image/png" sizes="32x32" href="/assets/logos/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/assets/logos/favicon-16x16.png">

<!-- Apple Touch Icon -->
<link rel="apple-touch-icon" sizes="180x180" href="/assets/logos/apple-touch-icon.png">

Features

  • Automatically adapts to light/dark system theme
  • SVG format for crisp display at any size
  • Simplified geometric icon from full logo

Generate PNG Versions

To create PNG fallbacks, you can use an online converter or ImageMagick:

# Using ImageMagick (if installed)
magick convert -background none -resize 32x32 favicon.svg favicon-32x32.png
magick convert -background none -resize 16x16 favicon.svg favicon-16x16.png
magick convert -background none -resize 180x180 favicon.svg apple-touch-icon.png
